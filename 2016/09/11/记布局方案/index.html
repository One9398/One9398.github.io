记网页布局方案

发表于 2016-09-11   |   分类于 编程   |  

水平居中的处理

方案一: inline-block+text-aligin

Note: text-align只针对当前元素下的子元素为inline/inline-block元素有效;inline-block元素的宽度根据内容决定.



方案二: table+margin

Note: display为table的元素行为上与block元素类似,但是宽度变为内容宽度;类似Block元素,margin:0 auto可以使得宽度确定的元素,自动计算左右margin而水平居中



方案三: absolute + transform

Note: 绝对定位元素的宽度也是基于自身内容确定;translateX的百分比值基于自身宽度计算;兼容性不支持IE8,替换方案为设置margin-left为自身一半宽度.



方案四: flex+justify-content

Note: display: flex的元素宽度就会自动计算;兼容性不支持IE8


垂直居中的处理

方案一: table-cell + vertical-align

Note: vertical-align针对行内元素以及table相关的元素有效.



方案二: absolute + transform

Note: translateY的百分比值基于自身高度计算;兼容性不支持IE8,替换方案为设置margin-top为自身一半高度.



方案三: flex + align-items

Note: display: flex元素的子元素高度默认拉伸为父元素高度,设置align-items为center时会取消子元素的拉伸行为.

水平垂直居中的处理

方案二: absolute + transform

方案三: flex + justify-content-align-items
