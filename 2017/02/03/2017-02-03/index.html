0203 ORID 学习日记

发表于 2017-02-03   |   分类于 学习日记   |  

第一次尝试下 ORID, 花了30分钟左右,感觉还是挺费时的.不过将一天自己的学习回顾和想法写下来的好处很快在持续一段时间后才会看到,坚持试试

Objective 客观事实

学了今天的课程.我记得什么

  • 老师介绍了JavaWeb 的课程体系,到最后会用 Servlet + JSP 做一个项目, 讲了 C/S 和 B/S 表示什么,各自有什么优缺点,当前主流时什么, 还讲了什么是 Tomcat 服务器, 并且如何安装启动配置 Tomcat 服务器,以及常见的问题;
  • 如何多种方式部署 Web 项目: 普通拷贝方式, 配置文件路径链接方式
  • 如何利用插件快速操作 Tomcat 服务器程序
  • 下午开始讲了HTTP 协议, 它是什么,有什么用,和一些关于请求和响应信息的内容介绍,以及 GET/POST 基本区别和简单使用

我完成了什么

  • 在电脑上安装好了 Tomcat服务器, 并且能运行 Web 项目
  • 给 Eclipse 安装了合适的 Tomcat 插件来快捷操作Tomcat服务器
  • 在Eclipse上创建一个普通 Java 工程,修改设置后成功部署了Web 项目
  • 安装了 Firework 浏览器,并且装了 FireBug 插件
  • 对这天课程的内容进行了简单的笔记总结

Reflective 我的感受

你要如何形容今天的情绪

感觉第一天的内容不多,掌握起来挺容易, 更多地是记忆和操作

今天的高峰是什么?

  • 下午在安装 Tomcat 插件后,开启Tomcat 服务器时打印台报错: 无权限访问日志文件, 通过Google 解决了并且进行了截图记录

今天的低点是什么?

  • 中午休息放松时有点焦虑, 不知道该怎么去休息

Interpretive 解释反思

我今天学到了什么?

  • 学到了两个高频小套路

今天一个重要的领悟是什么?

  • 遇到问题, 即使去搜索, 想办法自己解决, 不要依赖老师, 并且解决后要记录下来,帮助以后或者其他同学出现这问题时如何解决

Decisional 决定行动

我会如何用一句话形容今天的工作

今天学的内容没有代码, 要求了解和会操作偏多, 完成起来比较顺利; 但在假期阶段学习还是有点散漫,没能控制好时间

有哪些工作需要明天继续努力?

对 HTML/CSS 那些标签进行回顾下,代码实现他们的样式和属性,做好预习工作
